Stanford Dogs Classifier API

About

Welcome to the Stanford Dogs Classifier API documentation. This API allows you to classify dog breeds based on images of dogs.

Documentation

For detailed documentation on how to use the API, please refer to the following:

API Documentation

Usage

To use the API, you will need to provide authentication by log in to your account or one of the created as examples: admin, manager, user. After log in you will be able to upload images and make predictions using images/predict endpoint. Refer to the documentation for endpoint details and request parameters.
